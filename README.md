# Multimodal AI Analysis for Heart Disease Prediction
This project develops an AI-based system for predicting heart disease by combining clinical patient data and ECG signal analysis.

## Technologies Used
- Python
- Scikit-learn
- TensorFlow
- Keras
- Pandas
- NumPy
- Streamlit

## Models Used
- Random Forest for clinical data prediction
- CNN for ECG signal analysis
- Decision-level fusion for final prediction

## Features
- Predict heart disease risk using patient clinical data
- Upload ECG signal for analysis
- Multimodal prediction combining both models
- Interactive web interface using Streamlit
